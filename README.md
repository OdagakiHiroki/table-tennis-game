# table-tennis-game
## WIP
### できること
- 画面をグリグリするとカメラの位置を変えられる
- scrollでズームイン、アウト
- 右下のコントローラーでラケットの位置を変えられる
- 左下のアクションボタンでボールがトスされる（最初のみ）
- ボールをトスした後にもう一度アクションボタンを押すと、ラケットが煽っているかのように台の周りを回り始める
